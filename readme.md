## 📝 ¿Qué es Seiiverse?

**Seiiverse** es una plataforma de red social y microblogging que combina lo mejor de **Twitter** y **Reddit**, inspirada en el legendario **Miiverse** de Wii U y Nintendo 3DS.

Nacida de un proyecto de hardware (¡sí, estamos construyendo una Wii custom!), Seiiverse busca recuperar esa magia de las comunidades cerradas, amigables y llenas de creatividad que caracterizaban a la consola.

> 💬 *"Más que una red social, es un hogar digital para creadores y gamers"*
